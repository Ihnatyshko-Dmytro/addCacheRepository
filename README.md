# In-Memory Cache (Go)

Цей репозиторій містить реалізацію простого та потокобезпечного in-memory кешу на мові Go. Проєкт створено як виконання домашнього завдання (Тиждень 1).

## Особливості

* **In-memory зберігання**: Дані зберігаються в оперативній пам'яті за допомогою хеш-мапи (`map[string]interface{}`).
* **Потокобезпечність**: Безпечний конкурентний доступ до даних гарантується завдяки використанню `sync.RWMutex`.
* **Універсальність**: Кеш приймає будь-які типи даних як значення завдяки використанню порожнього інтерфейсу (`interface{}`).
* **Обробка помилок**: Метод отримання даних повертає зрозумілу помилку, якщо ключ не знайдено.

## Використання

Приклад використання пакету у вашому `main.go`:

```go
package main

import (
	"fmt"
	"log"
	"mycache/cache" // Замініть mycache на назву вашого модуля
)

func main() {
	// Ініціалізація нового кешу
	c := cache.NewCache()

	// Запис даних
	c.SetCache("user_1", "Олексій")
	c.SetCache("age", 25)

	// Отримання даних
	user, err := c.Get("user_1")
	if err != nil {
		log.Println(err)
	} else {
		fmt.Printf("Знайдено користувача: %v\n", user)
	}

	// Спроба отримати неіснуючий ключ
	_, err = c.Get("unknown_key")
	if err != nil {
		fmt.Printf("Очікувана помилка: %v\n", err)
	}

	// Видалення даних
	c.Delete("user_1")
}
